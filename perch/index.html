Everything your agents are doing, at a glance

Run three agents in parallel and you spend half your time cycling terminals to find the one that's blocked. Perch puts all of them in one glanceable place.

Live session list

Project, state (running / waiting / idle), last assistant message, token count, context-window gauge. Sessions appear within seconds of starting and vanish when their process dies.

Approve from the notch

When an agent asks to run a tool, a card pops with the tool name and pretty-printed input. ⌘A allow, ⌘N deny, ←→ walk the queue — your keystrokes land in Perch while your editor keeps focus.

Rate-limit gauges

Claude 5h/7d and Codex 5h/weekly windows with reset countdowns, plus a notification when you cross 80%.

Token usage dashboard

30 days of history: per day, per agent, per model, per project. Know where the budget went before the invoice does.

Why a watcher?

Ask an agent whether it's following your security rules and it will say yes. That answer costs nothing and proves nothing — the compromised case and the healthy case sound identical.

Perch stands outside the agent's process. What it shows isn't the agent's story about itself: session state, pending approvals, and token counts come from the harness's own records. The model can narrate whatever it likes; the record is written by the harness, not by the narration.

And a watcher only earns that seat if it can't become the next trust problem — which is why Perch is read-only, open source, and local-only. A watcher that phones home is just the trust problem wearing a new hat.

Install

Download the app

  1. Grab the latest .dmg from Releases and drag Perch into Applications.
  2. First launch: Perch is open-source and signed locally rather than notarized, so macOS asks once — approve it in Privacy & Security → Open Anyway.
  3. Menu bar → Install Claude Hooks… / Install Codex Hooks… (existing settings are parse-merged and backed up).
  4. Restart running sessions and verify with Doctor.

Build from source

Any Swift toolchain works — CommandLineTools is enough, no Xcode needed. No Gatekeeper dance either.

git clone https://github.com/theMobiusStrip/perch
cd perch
make run

Verify the DMG: each release ships a .sha256 and a GPG-signed checksum.
